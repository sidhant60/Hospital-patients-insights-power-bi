# Power BI Hospital Dashboard – Patient, Diagnosis & Billing Insights

## 📌 Overview
This Power BI dashboard analyzes hospital patient data, covering demographics, medical conditions, billing, and hospital operations.

## 📊 Key Features
- KPI cards for total patients, billing, length of stay
- Admissions trends and discharge breakdown
- Patient demographics by age, gender, and blood type
- Billing insights by insurance and condition
- Filters for hospital, doctor, and date

## 📁 Tools Used
- Power BI Desktop
- DAX (for KPIs and measures)
- Power Query for data cleaning
